# Drug Report Analyzer

Java application that analyzes forensic drug reports and determines whether samples pass testing criteria.

## Features

- Reads drug report files
- Processes marijuana, cocaine, and methamphetamine samples
- Performs multiple validation tests
- Uses custom exceptions for failed tests
- Generates separate pass and fail reports

## Skills Demonstrated

- Java
- Object-Oriented Programming
- File I/O
- Custom Exceptions
- Data Validation
- Collections
